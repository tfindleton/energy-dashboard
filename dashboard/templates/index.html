Compare solar generation, home usage, grid export, and grid import by day, ISO week, month, or year-to-date. Use the custom trend view to inspect seasonal drift, load shifts, and signs of reduced output over time.

This app uses the same local-browser flow as tesla-solar-download. Work through the steps below in order and the rest of the dashboard will unlock once local data exists.

1

Enter Tesla Account

Start with the Tesla account email. Site ID is optional and only needed if you want to pin one site.

2

Open Tesla Login

Use the generated Tesla sign-in link. You can open it directly or copy it somewhere else.

3

Paste Final Tesla URL

After Tesla finishes, copy the final URL from the expected Page Not Found page and paste it here.

The Tesla page will end on a Page Not Found screen at auth.tesla.com. That is expected for this local-only flow.
Once sign-in is complete, this app keeps the Tesla session token locally so future syncs only need to fetch new data.
Use local-time cron. Examples: 0 1 * * * daily at 1:00 AM, 0 6 * * 1-5 weekdays at 6:00 AM, or off to disable. Need help? Open the Crontap debugger.
